Dans la vidéo précédente, vous avez découvert Git et GitHub. Avant de poursuivre cette vidéo, créez un compte GitHub et connectez-vous. Commençons par créer un nouveau dépôt. Cliquez sur +, puis sur Nouveau référentiel. Pour créer un nouveau dépôt, vous devez fournir les informations suivantes : donner un nom à votre nouveau dépôt ; éventuellement, ajouter une description de votre dépôt ; choisir la visibilité du dépôt (si vous voulez qu'il soit public ou privé) et choisir l'option Initialiser ce dépôt avec un fichier readme. Cliquez ensuite sur Créer un référentiel. Vous allez maintenant être redirigé vers le référentiel que vous avez créé. Le dossier racine de votre dépôt est répertorié par défaut et il ne contient qu'un seul fichier README.md. Il est maintenant temps de modifier le fichier readme. Vous pouvez le faire dans votre navigateur. Cliquez simplement sur le crayon pour ouvrir l' éditeur en ligne et vous pouvez modifier le texte du fichier readme. Pour enregistrer les modifications apportées au référentiel, vous devez les valider. Une fois que vous avez apporté vos modifications, faites défiler la page vers le bas jusqu'à la section Valider les modifications. Ajoutez un message de validation et ajoutez éventuellement une description, puis cliquez sur Valider les modifications. La « validation des modifications » est utilisée pour enregistrer vos modifications dans le référentiel. Revenez à l'écran d'accueil en cliquant sur le lien du nom du dépôt. Notez que le fichier readme est mis à jour et vérifiez vos modifications. Apprenons à créer un nouveau fichier à l'aide de l'éditeur Web intégré fourni par GitHub qui s'exécute dans le navigateur. Cliquez sur Ajouter un fichier, puis sur Créer un nouveau fichier pour créer le nouveau fichier. Pour créer un fichier python appelé firstpython.py. Indiquez d'abord le nom du fichier. Ajoutez ensuite un commentaire qui décrit votre code, puis ajoutez-le. Une fois terminé, validez la modification dans le référentiel. Vous pouvez constater que votre fichier est désormais ajouté au référentiel et la liste des référentiels indique quand le fichier a été ajouté ou modifié. Lorsque vous devez modifier le fichier, vous pouvez le modifier à nouveau. Cliquez sur le nom du fichier, puis sur l'icône en forme de crayon, apportez vos modifications et validez les modifications. Vous pouvez également télécharger un fichier depuis votre système local dans le référentiel. Sur l'écran d'accueil du référentiel, cliquez sur Ajouter un fichier et choisissez l'option Charger des fichiers. Cliquez sur Choisissez vos fichiers et sélectionnez les fichiers que vous souhaitez télécharger depuis votre système local. Le processus de téléchargement de fichiers peut prendre un peu de temps, en fonction de ce que vous êtes en train de télécharger. Une fois le téléchargement des fichiers terminé, cliquez sur Valider les modifications. Le référentiel reflète désormais les fichiers qui ont été téléchargés. Dans cette vidéo, vous avez appris à créer un référentiel, à modifier des fichiers et à valider des modifications à l'aide de l'interface Web.
